## إثراء القوالب وجعلها تبدو غنية واحترافية 🎨

> **المشكلة الحالية**: القوالب حالياً تعرض Hero + مشاريع + تواصل فقط. الموقع يبدو بسيط وفارغ،
> وهذا لا يعطي انطباعاً احترافياً لعملاء المكتب. كل قالب يحتاج محتوى أكثر وأقسام متنوعة.

### أقسام يجب إضافتها لكل قالب

#### ١. شريط علوي (Top Bar) - أعلى الهيدر
- رقم الجوال
- البريد الإلكتروني
- أيقونات السوشيال ميديا
- يظهر في كل الصفحات

#### ٢. الهيدر (Header) - محسّن
- الشعار بحجم مناسب
- القائمة: الرئيسية | من نحن | خدماتنا | مشاريعنا | تواصل معنا
- يكون Sticky (يثبت عند scroll)
- تأثير شفافية عند التمرير

#### ٣. Hero Section - محسّن
- صورة غلاف كبيرة بجودة عالية (يفضل فيديو قصير)
- عنوان رئيسي قوي + عنوان فرعي
- شعار المكتب بشكل بارز
- إحصائيات سريعة (عدد المشاريع، سنوات الخبرة، عدد العملاء)
- زران: "شاهد مشاريعنا" + "تواصل معنا"
- تأثيرات حركية خفيفة (Fade in, Slide up)

#### ٤. قسم "من نحن" (About Section)
- نص النبذة التعريفية من بيانات المكتب
- صورة المكتب أو صورة تعبر عنه
- ممكن إضافة mission / vision / values
- أيقونات تشرح رؤية المكتب ورسالته
- إحصائيات (أرقام): عدد المشاريع المنجزة، سنوات الخبرة، عدد العملاء، الموظفين

#### ٥. قسم الخدمات (Services)
- عرض الخدمات الهندسية اللي يقدمها المكتب:
  - تصميم معماري
  - تصميم داخلي
  - إشراف على التنفيذ
  - استشارات هندسية
  - تخطيط عمراني
  - إدارة مشاريع
- كل خدمة: أيقونة + اسم + وصف مختصر
- ترتيب Grid جميل (3 أو 4 أعمدة)

#### ٦. قسم "لماذا تختارنا" (Why Us)
- نقاط القوة والمميزات:
  - خبرة XX سنة
  - فريق مهندسين معتمدين
  - التزام بالمواعيد
  - أسعار تنافسية
  - متابعة مستمرة
  - تصاميم مبتكرة
- كل نقطة مع أيقونة جميلة

#### ٧. قسم المشاريع المميزة (Featured Projects) - محسّن
- ليس مجرد Grid صور فقط
- كل مشروع: صورة كبيرة + اسم المشروع + تصنيف + سنة + موقع
- Hover effect: الصورة تتوسع، يظهر وصف مختصر
- فلتر حسب التصنيف (سكني، تجاري، صناعي...)
- زر "شاهد كل المشاريع" في الأسفل
- Carousel/Slider للمشاريع المميزة في أعلى الصفحة

#### ٨. قسم الأرقام والإحصائيات (Counter Section)
- عدد المشاريع المنجزة (يتحرك العداد من 0 إلى الرقم)
- سنوات الخبرة
- عدد العملاء
- عدد الموظفين / المهندسين
- تأثير حركي للعدادات عند الظهور (Intersection Observer)

#### ٩. قسم فريق العمل (Team)
- صور أعضاء الفريق
- الاسم، المنصب، الوصف
- روابط السوشيال ميديا (لينكدإن)
- تصميم Grid أو Carousel

#### ١٠. قسم شهادات العملاء (Testimonials)
- رأي العميل + اسمه + منصبه + صورته
- تقييم بالنجوم (⭐⭐⭐⭐⭐)
- تصميم Carousel يتنقل تلقائياً
- يكون بصيغة اقتباس جميل

#### ١١. قسم العملاء / الشركاء (Clients / Partners)
- شعارات الشركات أو الجهات اللي تعاملت معهم
- يكون بشكل شريط لوقوهات يتحرك
- يعطي ثقة ومصداقية

#### ١٢. قسم "قيد الإنشاء" (Under Construction)
- عرض المشاريع اللي تحت التنفيذ
- صورة + اسم المشروع + نسبة الإنجاز (%)
- Progress bar جميل

#### ١٣. قسم الأسئلة الشائعة (FAQ)
- أسئلة وأجوبة عن خدمات المكتب
- تصميم Accordion (يفتح ويقفل)
- أسئلة مثل: كم مدة التصميم؟ هل تقدمون استشارات؟ إلخ

#### ١٤. قسم دعوة للتواصل (CTA - Call to Action)
- نص تشجيعي: "هل عندك مشروع؟ دعنا نناقشه"
- زر "تواصل معنا الآن" + زر واتساب
- خلفية جذابة (لون متدرج أو صورة)
- يظهر قبل الفوتر

#### ١٥. الفوتر (Footer) - محسّن وغني
- 3-4 أعمدة:
  - العمود ١: الشعار + نبذة مختصرة
  - العمود ٢: روابط سريعة (الرئيسية، المشاريع، من نحن، تواصل)
  - العمود ٣: معلومات التواصل (جوال، إيميل، عنوان)
  - العمود ٤: تابعنا (أيقونات السوشيال ميديا)
- خريطة الموقع مصغرة
- حقوق النشر
- رابط "صُمم بواسطة [اسم المنصة]"

- [ ] **Breadcrumb** مسار التنقل (الرئيسية > المشاريع > اسم المشروع)
- [ ] **تأثيرات حركية خفيفة** (Fade, Slide, Scale) عند scroll
- [ ] **Shimmer/Skeleton loading** بدل شاشة بيضاء أثناء التحميل
- [ ] **Badge على المشاريع**: "مميز"، "جديد"، "قيد الإنشاء"
- [ ] **مشاركة المشروع**: أزرار واتساب، تويتر، لينكدإن في صفحة المشروع
- [ ] **قراءة المزيد**: نص طويل يختصر مع زر "عرض المزيد"
- [ ] **Lightbox للصور**: تكبير الصور مع سلايدر في صفحة المشروع
- [ ] **زرار اتصال سريع** (Click to call) في الجوال
- [ ] **Video Background في الـ Hero** (اختياري)
- [ ] **تغيير لون الهيدر عند التمرير** (شفاف ← معتم)
- [ ] **عداد تصاعدي** في قسم الإحصائيات

- [ ] **صورة Before/After** للمشاريع (قبل وبعد)

### معالجة مشكلة "الموقع الفارغ"

> عندما يسجل مكتب جديد، لا توجد لديه بيانات كافية. يجب التعامل مع هذه الحالة بذكاء:

- [ ] **بيانات افتراضية ذكية**: إذا لم يحدد صورة غلاف → عرض gradient جميل مع اسم المكتب
- [ ] **نصوص بديلة**: إذا لم يكتب نبذة → عرض نص افتراضي: "نحن مكتب هندسي متخصص في..."
- [ ] **إخفاء الأقسام الفارغة**: إذا لا توجد بيانات لقسم معين، لا يظهر أبداً
- [ ] **رسائل تشجيعية في الداشبورد**: "أضف نبذة عن مكتبك ليكتمل موقعك"
- [ ] **نسبة اكتمال الموقع**: يظهر للمكتب كم أكمل من ملفه (%)
- [ ] **زر "عرض الكل"**: إذا المشاريع أكثر من 6 → زر يوصل لصفحة المشاريع
- [ ] **التعامل مع الصور المفقودة**: صورة افتراضية بديلة للمشاريع اللي بدون صور
- [ ] **Empty State جميل**: إذا ما فيه مشاريع → "قريباً سيتم إضافة مشاريعنا" مع تصميم جميل

## داشبورد المكتب

- [ ] **إحصائيات متقدمة**: عدد زيارات الموقع، أكثر المشاريع مشاهدة
- [ ] **خدمات المكتب**: إضافة قسم يعرض الخدمات الهندسية اللي يقدمها المكتب
- [ ] **فريق العمل**: إضافة أعضاء الفريق (اسم، صورة، منصب، وصف)
- [ ] **شهادات العملاء**: آراء وتقييمات العملاء تظهر في الموقع
- [ ] **المدونة / المقالات**: نشر مقالات وأخبار عن المشاريع
- [ ] **جدولة المواعيد**: نموذج حجز استشارة أو اجتماع
- [ ] **معرض فيديو**: رفع فيديوهات للمشاريع (YouTube/Vimeo أو رفع مباشر)
- [ ] **تصدير البيانات**: تصدير المشاريع كـ PDF أو جدول
- [ ] **إحصائيات التخزين**: معرفة المساحة المستخدمة والمتبقية
- [ ] **نسخ احتياطي**: تصدير كامل لبيانات المكتب

## موقع المكتب العام

- [ ] **نموذج تواصل**: فورم اتصال يرسل إيميل لصاحب المكتب
- [ ] **بحث في المشاريع**: شريط بحث عن المشاريع بالاسم أو التصنيف
- [ ] **فلترة متقدمة**: فلترة المشاريع حسب السنة، التصنيف، المدينة
- [ ] **خريطة تفاعلية**: عرض مواقع المشاريع على خريطة (Leaflet/Google Maps)
- [ ] **مشاركة المشاريع**: أزرار مشاركة المشروع في واتساب، تويتر، لينكدإن
- [ ] **معرض صور كامل**: Lightbox متطور مع تكبير وسلايدر
- [ ] **صفحة "قيد الإنشاء"**: للمشاريع اللي تحت التنفيذ
- [ ] **وضع Dark Mode**: يتغير تلقائياً حسب إعدادات الجهاز
- [ ] **Chatbot بسيط**: ردود تلقائية على الأسئلة الشائعة
- [ ] **شهادة الاعتماد**: عرض شهادات ورخص المكتب الهندسية

## لوحة تحكم الأدمن

- [ ] **لوحة تحليلات شاملة**: رسوم بيانية لإحصائيات المنصة كاملة
- [ ] **إشعارات تلقائية**: إشعارات داخل الداشبورد للمهام العاجلة
- [ ] **تصدير التقارير**: تقارير Excel/PDF للمكاتب والاشتراكات
- [ ] **إدارة الباقات**: إضافة/تعديل/حذف الباقات والأسعار من الداشبورد
- [ ] **نظام إشعارات جماعي**: إرسال إيميل لجميع المكاتب دفعة واحدة
- [ ] **سجل النشاطات**: Audit log لمراقبة إجراءات الأدمن
- [ ] **إدارة القوالب**: تفعيل/تعطيل قوالب معينة
- [ ] **إدارة الصلاحيات**: أدمن متعدد بصلاحيات مختلفة
- [ ] **إعدادات عامة**: تعديل رقم الواتساب، نص الإيميلات من الداشبورد
- [ ] **حذف الحسابات تلقائياً**: جدولة حذف الحسابات المنتهية بعد فترة

## نظام الاشتراك والدفع

- [ ] **ربط بوابة دفع**: Stripe / PayTabs / Moyasar للمدفوعات
- [ ] **فواتير تلقائية**: إصدار فاتورة إلكترونية مع كل تجديد
- [ ] **تجديد تلقائي**: اشتراك شهري/سنوي يتجدد تلقائياً
- [ ] **فترة تجريبية مجانية**: 14 يوم مجاناً للمكاتب الجديدة
- [ ] **كوبونات خصم**: إنشاء أكواد خصم من الأدمن
- [ ] **ترقية/تخفيض الباقة**: ترقية فورية مع احتساب فرق السعر
- [ ] **إشعارات الدفع**: تنبيه قبل التجديد بـ 30، 14، 7، 1 يوم
- [ ] **سجل الفواتير**: عرض وتحميل الفواتير السابقة

- [ ] **قوالب إيميل HTML**: تصميم إيميلات احترافية بشعار المنصة
- [ ] **إيميل تأكيد البريد**: تأكيد البريد الإلكتروني عند إنشاء الحساب

- [ ] **إيميل ترحيب مخصص**: رسالة ترحيب باسم المكتب
- [ ] **إيميل أسبوعي**: إحصائيات الزيارات الأسبوعية
